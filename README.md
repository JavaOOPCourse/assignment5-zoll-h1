## Task 1 — Count Vowels
Write a Java program that reads a string from the user and counts the number of vowels in the string (a, e, i, o, u).

**Задание 1 — Подсчёт гласных**  
Напишите программу на Java, которая считывает строку от пользователя и подсчитывает количество гласных букв в строке (a, e, i, o, u).

## Task 2 — Reverse a String
Write a program that reads a string and prints the string in reverse order.

**Задание 2 — Разворот строки**  
Напишите программу, которая считывает строку и выводит её в обратном порядке.

## Task 3 — Check Palindrome
Write a program that checks whether a string is a palindrome (reads the same forward and backward). Ignore case.

**Задание 3 — Проверка на палиндром**  
Напишите программу, которая проверяет, является ли строка палиндромом (читается одинаково слева направо и справа налево). Игнорируйте регистр букв.

## Task 4 — Count Words in a Sentence
Write a program that reads a sentence and counts how many words it contains.

**Задание 4 — Подсчёт слов в предложении**  
Напишите программу, которая считывает предложение и подсчитывает, сколько слов оно содержит.

## Task 5 — Remove All Spaces
Write a program that removes all spaces from a given string.

**Задание 5 — Удаление всех пробелов**  
Напишите программу, которая удаляет все пробелы из заданной строки.

## Task 6 — Capitalize First Letter of Each Word
Write a program that converts the first letter of each word in a sentence to uppercase.

**Задание 6 — Заглавная первая буква каждого слова**  
Напишите программу, которая преобразует первую букву каждого слова в предложении в заглавную.

## Task 7 — Find the Most Frequent Character
Write a program that finds the most frequently occurring character in a string.

**Задание 7 — Поиск самого частого символа**  
Напишите программу, которая находит символ, который встречается чаще всего в строке.

## Task 8 — Check Anagrams
Write a program that checks whether two strings are anagrams (contain the same characters in different order). Ignore spaces and letter case.

**Задание 8 — Проверка анаграмм**  
Напишите программу, которая проверяет, являются ли две строки анаграммами (содержат одни и те же символы, но в разном порядке). Игнорируйте пробелы и регистр букв.
